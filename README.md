                                    ТЕЛЕФОННЫЙ СПРАВОЧНИК

Программный инструмент для управления контактами: хранения, поиска, редактирования и удаления записей.
Данные сохраняются в локальный файл.

Описание
Цель проекта:
Создать приложение для ведения телефонного справочника.

Основные функции:
Открытие файла с контактами.
Сохранение изменений в файл.
Просмотр всех сохранённых контактов.
Добавление нового контакта.
Поиск контакта.
Редактирование существующих записей.
Удаление контактов.
Безопасный выход из программы.

Установка:
Убедитесь, что на вашем компьютере установлен Python 3.x (рекомендуется версия 3.8 и выше).
Скачайте репозиторий проекта или скопируйте файлы в удобную директорию.
Откройте терминал (командную строку) в папке с проектом и введите.
Запустите программу:
python main.py

Использование
После запуска программы вы увидите главное меню с номерами команд.
Выберите нужное действие, введя соответствующую цифру:

1. Показать все контакты — выводит список всех контактов.
2. Добавить контакт — введите имя, номер телефона и, при необходимости, комментарий.
3. Удалить контакт — выберите контакт для удаления по имени или номеру или комментарию.
4. Редактировать контакт — найдите контакт и обновите его поля (имя, номер, комментарий).
5. Найти контакт — укажите имя или номер или комментарий для поиска; программа покажет совпадающие записи.
6. Выход — завершение работы программы.
